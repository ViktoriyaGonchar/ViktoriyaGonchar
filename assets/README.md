# Папка для скриншотов проектов

Эта папка предназначена для хранения изображений, используемых в основном README.md.

## Необходимые изображения:

### SHOP SHIPS
- `shop-ships-main.png` - Скриншот главной страницы интернет-магазина
- `shop-ships-admin.png` - Скриншот админ-панели CRM

### Security Scanner API
- `security-scanner-main.png` - Скриншот главной страницы с формой сканирования
- `security-scanner-admin.png` - Скриншот админ-панели с историей сканирований

## Размеры изображений (рекомендуемые):
- Ширина: 800-1200px
- Формат: PNG или JPG
- Размер файла: до 500KB (для быстрой загрузки на GitHub)

## Как добавить изображения:
1. Сделайте скриншоты ваших проектов
2. Переименуйте файлы согласно списку выше
3. Поместите их в эту папку
4. Изображения автоматически отобразятся в основном README.md
